# Tareas de apropiación javascript


JULIAN:

1. Saludo personalizado
• Solicita al usuario su nombre y edad.
• Muestra en consola: "Hola [nombre], tienes [edad] años."
2. Área de un rectángulo
• Solicita base y altura.
• Calcula y muestra el área.
3. Promedio de tres notas
• Solicita tres calificaciones.
• Calcula y muestra el promedio.

SARA:

4. Conversión de monedas
• Solicita un valor en pesos colombianos.
• Convierte a dólares (suponiendo 1 USD = 4000 COP).

5. Conversión de temperatura
• Solicita una temperatura en Celsius.
• Convierte a Fahrenheit usando la fórmula: F = (C × 9/5) + 32.

• Calcula el perímetro y muéstralo.
